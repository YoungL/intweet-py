Welcome to InTweet

Twitter Sentiment Analysis Tool

InTweet is a sentiment analysis tool which allows users to effectively monitor the general user perception of publicly posted tweets about a company on Twitter. InTweet is a fully automated tool, allowing the user to input monitoring rules which are then automatically harvested from Twitter on a regular basis and analysed using our sentiment detection algorithm.
